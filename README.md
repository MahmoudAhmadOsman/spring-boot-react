## Java React Full Stack 

### Project Description

This product is based on Java and React js.



### Technologies

**Persistence Tier**
- PostGreSQL (running on Docker)

**Application Tier**
- Written in Java 8.
- Intellij
- Amazon AWS
- Apache Maven
- Spring Boot (Backend)
- React JS (Frontend)
- JSON Web Tokens
- JUnit
- Mockito
- POSTMAN
